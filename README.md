 # Healthcare Data Analysis

## Overview
This project analyzes a synthetic healthcare dataset using Python, Pandas, and Matplotlib. The objective was to explore patient demographics, medical conditions, billing trends, and laboratory test results through exploratory data analysis (EDA).

## Tools
- Python
- Pandas
- Matplotlib

## Dataset
- 55,500 patient records
- 15 variables
- No missing values

## Project Objectives
- Explore patient demographics
- Analyze medical conditions
- Examine billing trends
- Visualize healthcare data

## Visualizations
- Medical Condition Bar Chart
- Gender Distribution Pie Chart
- Billing Amount Histogram
- Correlation Heatmap

## Key Findings
- The dataset contains 55,500 patient records and 15 variables.
- There are no missing values.
- Male and female patients are almost equally represented.
- Medical conditions occur at similar frequencies.
- Billing amounts are widely distributed.
- There is very little correlation between Age, Billing Amount, and Room Number.
